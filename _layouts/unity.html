Server Surfer

Coding, Art, & Music by Joshua "Tello-" Lollis

Controls:

Activate Controls: [Click Game Area]

Change Stance: [LEFT], [RIGHT]

Lift Board: [UP], [SPACE]

Halt Board: [DOWN]

Synopsis:

Surf the server-verse as the Server Surfer! A good surfer should alter their stance, allowing for different ways to carve the waves!

How to win:

Have too much fun!

The game will determine when you've had too much fun. Once the win state is detected, the game will inform you. So keep trying for the ultimate victory, fun!

Notes:

This is a prototype. My goal with this prototype was to create a few new mechanics for a flappy-like. The stance position on the board affects the surfer's lift-behavior. In development mode (Unity Editor), designers can hotswap enemy flight patterns at the drop of a pattern object. This game prototype utilizes concepts like dependency injection, custom events/triggers, color lerping, etc. This build does not factor in collisions between obstacles and the player. The main showcase of this build is the surfer control mechanics, and the enemy movement dependency injection aspect. The art and music is by myself. The music uses samples form other sources, but is soley used here for education purposes. Nothing from this project, or on this website is for sale, ever.
